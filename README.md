## Fractal 
A simple version of a **Fractal** using the **MLX** (MiniLibX) library in C.

![qqq](https://github.com/user-attachments/assets/a4fc2459-d6c2-47bb-bbb2-9564c241450f)

## How to run
	git clone https://github.com/vahe4060/Fractal.git
	cd Fractal
	make
    ./fractal 
    # Usage: ./fractal [0-2]
    #    [0] - julia set
    #    [1] - mondelbrod set
    #    [2] - burning ship set
